# TERA-tail-light

Custom automotive tail light controller based on the STM32F411CEU6: five
opto-isolated, PDM/PWM-tolerant vehicle inputs (running/brake/turn/reverse/
aux) driving two WS2812B strips via TIM1 + DMA.

Firmware, wiring notes and the STM32CubeMX setup guide live in
[`firmware/`](firmware/README.md).
